Card Counting

This is where the exciting part

Reasons:

  • Using the HI/LO system
  • Cards: 2-6 are considered positive cards
  • Cards: 7-9 are neutral you ignore these in a sense for counting
  • Cards: 10,J,Q,K,A are negative cards

This is just some simple information about HI/LO system
